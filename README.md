# Project Obsidian

Personal digital garden and braindump, built with Quartz v4.
Evolving notes intended for mah personal brain-dump and side projects.

## Local use
- Install: `npx -p npm@10.9.2 npm install`
- Build: `node ./quartz/bootstrap-cli.mjs build`
- Preview: `node ./quartz/bootstrap-cli.mjs build --serve`

## Structure
- `content/` notes and pages
- `content/AI_Sandbox/` drafts and experiments
- `content/attachments/` assets
