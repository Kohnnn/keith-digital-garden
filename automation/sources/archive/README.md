# X Link Archives

This folder stores archived weekly X link sets after processing.

Structure:

- `automation/sources/archive/<year>/<year>-W<week>-x-posts.md`

Example:

- `automation/sources/archive/2023/2023-W01-x-posts.md`

These files are generated by:

```bash
python "automation/scripts/process_weekly_x_links.py" --output-dir "content/attachments/market/2023/W01" --archive-input --clear-temp-after-archive
```
