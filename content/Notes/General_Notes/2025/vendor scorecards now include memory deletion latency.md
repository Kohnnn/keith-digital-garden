# vendor scorecards now include memory deletion latency

Enterprise governance teams are adding memory deletion latency to vendor reviews as retention obligations tighten and revocation windows shorten ([NIST privacy framework](https://www.nist.gov/privacy-framework)).

*see also:* [[enterprise memory retention rules move into legal policy]] · [[dataset revocation clauses force faster index hygiene]]

## control evolution
Deletion capability is now measured as an operational KPI instead of a policy statement.

## signal braid
- Contracts increasingly specify deletion response windows.
- Audit scope extends to derived memory artifacts.
- Vendors with automated deletion paths gain trust advantage.

## my take
Deletion latency is becoming one of the clearest signals of real governance maturity.
